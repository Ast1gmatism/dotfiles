Wayland-сетап с боковой панелью, попапами и SDDM-темой на QML.
Конфиг Hyprland написан на Lua.

---

## Что уже работает

- Сайдбар с виджетами — часы, воркспейсы, трей с контекстным меню, громкость, батарея, bluetooth, сеть
- Power popup — уровень заряда, профили питания, яркость, redshift
- OSD — появляется при изменении громкости и яркости
- **Динамическая тема** — цвета подхватываются из `colors.json` без перезапуска
- Кастомная SDDM-тема
