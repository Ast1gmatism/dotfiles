# 🌸 Hyprland & Quickshell Dotfiles (Alpha)

Минималистичный сетап для Wayland на базе **Hyprland** и **Quickshell**.
Сайдбар, попапы и SDDM тема написаны на QML.

> [!WARNING]
> ⚠️ **Статус: Alpha (v0.x.x)**
> Проект находится в активной разработке. Конфигурация может меняться.
> Любая помощь, баг-репорты и пулл-реквесты приветствуются!

## Структура
- `configs/quickshell` — Сайдбар, виджеты, попапы (QML).
- `configs/hypr` — Конфигурация Hyprland (lua).
- `configs/sddm` — Кастомная тема для SDDM.

## Установка
Конфиги привязаны к `~/.config` через симлинки.
